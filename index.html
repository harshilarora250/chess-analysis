FAQ

Is ChessCalc really free?
Yes, completely. No hidden fees, no feature locks, no account required. We believe chess analysis should be accessible to everyone.
What does the supporter badge look like?
A ⭐ next to your username when we add user accounts. For now it's more of a spiritual badge while we build that out.
What if you shut down?
ChessCalc runs entirely in your browser — the code is open. Even if we disappeared, anyone could host it.
Do you show ads?
No. ChessCalc is ad-free and we intend to keep it that way.
